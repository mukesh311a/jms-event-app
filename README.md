# JMS Event — AppSheet kit

**Everything you need is in one place.**

1. Run `python scripts/setup_excel.py` to build **`JMS-Event-Data.xlsx`**.  
2. Upload that file to cloud and create your AppSheet app from it.  
3. Follow **`START-HERE.md`** end-to-end.  
4. Use **`APPSHEET-FULL-SPEC.md`** for all formulas and validation text.

Supporting files: **`data/*.csv`** (sources), **`templates/`** (ID + invitation wording for PDF bots).
